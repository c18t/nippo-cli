# nippo-cli
The tool to power my nippo.

## Install `nippo` command
```
go install github.com/c18t/nippo-cli/nippo@latest
```

## Usage
### Setup
```console
nippo init
```

### Build
```console
nippo build
```

### Publish
```console
nippo deploy
```

## Setting up your development environment
```
> docker compose up -d
> docker compose exec nippo-cli bash
$ go run nippo/nippo.go
$ make
```
