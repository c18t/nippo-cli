# nippo-cli
The tool to power my nippo.

## Usage
### Setup
```console
nippo init
```

### Build
```console
nippo build
```

### Publish
```console
nippo deploy
```

## Setting up your development environment
```
> docker compose up -d
> docker compose exec nippo-cli bash
$ go run nippo.go
```
